# Visão Geral

Este repositório reúne documentação e scripts relacionados ao projeto `PDV_MAX`.

## O que entra aqui

- orientações de instalação;
- configurações iniciais;
- ajustes operacionais;
- anotações de versão e mudança.

## Convenção

- mudanças pequenas e compatíveis aumentam a versão de patch;
- novas funcionalidades compatíveis aumentam a versão de minor;
- mudanças incompatíveis aumentam a versão de major.
