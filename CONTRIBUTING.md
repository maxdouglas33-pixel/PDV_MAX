# Contributing

Obrigado por ajudar no `PDV_MAX`.

## Como contribuir

1. Crie uma branch com nome descritivo.
2. Faça alterações pequenas e focadas.
3. Atualize a documentação relacionada.
4. Registre mudanças em `CHANGELOG.md` quando necessário.
5. Verifique se os scripts continuam executáveis.

## Padrão de versionamento

- correções pequenas: `PATCH`;
- novas funcionalidades compatíveis: `MINOR`;
- mudanças incompatíveis: `MAJOR`.

## Boas práticas

- mantenha arquivos e documentação sincronizados;
- evite mudanças sem contexto;
- prefira mensagens de commit objetivas;
- preserve a simplicidade do fluxo inicial.
