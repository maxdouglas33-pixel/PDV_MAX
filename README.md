# PDV_MAX

Repositório oficial de documentação, scripts de base e versionamento do projeto.

## Objetivo

Centralizar:

- documentação técnica e operacional;
- scripts de instalação/configuração;
- histórico de mudanças por versão;
- base para releases futuras no GitHub.

## Estrutura

- `docs/` - documentação do sistema e do processo de versionamento.
- `scripts/` - automações e scripts de apoio.
- `CHANGELOG.md` - histórico de alterações.
- `VERSION` - versão atual do repositório.

## Versão atual

`0.1.0`

## Como usar

1. Leia [`docs/visao-geral.md`](docs/visao-geral.md) para entender a organização do projeto.
2. Consulte [`docs/versionamento.md`](docs/versionamento.md) para a regra de versões.
3. Execute os scripts em `scripts/` conforme a necessidade do ambiente.
